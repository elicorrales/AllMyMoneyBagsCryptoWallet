Vaultlet

A local-first, secure-by-default crypto wallet built for the desktop.

🌐 Overview

Vaultlet is a standalone desktop crypto wallet designed to prioritize privacy, control, and offline access. It avoids the cloud, runs on file://, and communicates via a secure local proxy.

🛠 Features

  • Full dApp compatibility via injected JS (EVM-compatible)
  • Local proxy server for secure RPC networking
  • File:// protocol support for airgapped/local deployments
  • Modular codebase: separate logic for RPC, signing, and UI
  • Works with Stellar testnet and EVM chains

🧠 Sample Code


// injected-dapp.js (simplified)
;(function () {
  const provider = {
    isVaultlet: true,
    request: async ({ method, params }) => {
      // Sends request through local proxy
      return sendRpcViaProxy(method, params);
    }
  };
  window.ethereum = provider;
})();
